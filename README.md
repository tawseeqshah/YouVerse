# YouVerse

YouVerse is a comprehensive React-based mobile application that combines various utilities and features to create a personalized digital universe for users.

## Features

- 🕌 Prayer Times
- ⛅ Weather Information
- 📖 Daily Hadith
- ✅ Todo Management
- 📝 Notes
- 👥 Contacts
- 🎨 Theme Customization
- 📍 Location-based Services

## Tech Stack

- React
- Capacitor
- Firebase
- Framer Motion
- React Router
- Axios
